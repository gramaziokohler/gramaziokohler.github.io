class compas_fab.robots.Robot(robot_model, robot_artist, semantics=None, client=None)[source]

Bases: object

Represents a robot instance.

This class binds together several building blocks, such as the robot’s descriptive model, its semantic information and an instance of a backend client into a cohesive programmable interface. This representation builds upon the model described in the class compas.robots.RobotModel of the COMPAS framework.

model

compas.robots.RobotModel – The robot model, usually created out of an URDF structure.

artist

compas_fab.artists.BaseRobotArtist – Instance of the artist used to visualize the robot.

semantics

RobotSemantics, optional – The semantic model of the robot.

client

optional – The backend client to use for communication, e.g. RosClient

name

str – The name of the robot
