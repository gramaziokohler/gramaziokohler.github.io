class compas_fab.robots.Robot(robot_model, semantics=None, client=None)[source]

Bases: object

Represents a robot instance.

This class binds together several building blocks, such as the robot’s descriptive model, its semantic information and an instance of a backend client into a cohesive programmable interface. This representation builds upon the model described in the class compas.robots.Robot of the COMPAS framework.

robot_model

compas.robots.Robot – The robot model, usually created out of an URDF structure.

semantics

RobotSemantics, optional – The semantic model of the robot.

client

optional – The backend client to use for communication, e.g. RosClient

name

str – The name of the robot
