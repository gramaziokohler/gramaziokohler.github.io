BaseRobotArtist.update_link

BaseRobotArtist.update_link(link, joint_state, parent_transformation, collision=True)[source]

Recursive function to apply the transformations given by the joint state.

The parameter joint_state is given as absolute values, so it is necessary to reset the current transformation.

Parameters

  • link (compas.robots.Link) – Link instance to update.

  • joint_state (dict) – A dictionary with the joint names as keys and values in radians and meters (depending on the joint type).

  • parent_transformation (Transformation) – The transfomation of the parent joint.

  • collision (bool) – True if the collision geometry should be also updated, otherwise False. Defaults to True.
